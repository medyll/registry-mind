## Technical Notes

**HoldToTalkButton States:**
| State | Color | Animation |
|-------|-------|-----------|
| IDLE | Blue (#2196F3) | Pulse scale (1.0 → 1.1) |
| LISTENING | Red (#F44336) | Waveform bars (5 bars) |
| PROCESSING | Yellow (#FFC107) | Spinner rotation (270° arc) |

**Waveform Animation:**
- 5 vertical bars with gradient (light blue → blue)
- Animated height based on progress
- 800ms cycle with EaseInOut

**Session Timeout:**
- 30 seconds of inactivity
- Auto-ends session, resets state
- Persists across app restarts (EncryptedSharedPreferences)

**Haptic Patterns:**
- Capture initiated: 50ms single tick
- Sync successful: 50ms + 30ms delay + 50ms (double pulse)
- Error: 200ms long vibration
- Voice start/end: 30ms / 20ms subtle feedback
