## Security Features

**Encrypted Storage:**
- `EncryptedSharedPreferences` with AES256-GCM
- `MasterKey` with AES256-GCM key scheme
- Auth token never stored in plaintext

**Network Security:**
- Configurable timeout (default 30s)
- Exponential backoff on retry
- Connection pooling via OkHttpClient

## Sync Behavior

**Periodic Sync:** Every 15 minutes when network connected
**One-Time Sync:** Triggered by OpenClawReceiver ACTION_SYNC/FLUSH
**Backoff:** Exponential, starting at 1 minute
