## Technical Notes

**Overlay Types:**
1. **Peripheral Glow** - Full-screen glassmorphic border (500ms auto-dismiss)
2. **Liquid Button** - Draggable floating button (anchored vs floating modes)

**Snap Key Detection:**
- Uses AccessibilityService API
- Detects TYPE_VIEW_CLICKED events
- Long press detection via gesture timeout

**Service Architecture:**
- `CaptureService` runs as foreground service
- Notification channel with low importance
- STICKY return type for service restart
