## Pipeline Flow

```
Snap Key Press
    ↓
CaptureService.startCapture()
    ↓
RegistryIngestor.captureAndSend()
    ↓
ScreenCapturer.capture() → Bitmap
    ↓
OcrProcessor.process() → OCR text
    ↓
AppContextScraper.getCurrentPackage() → Package name
    ↓
RegistryPacket assembled
    ↓
ClawConnector.sendPacket() → HTTP POST
    ↓
Success / Cache for retry
```

## Technical Notes

**ML Kit Configuration:**
- Uses Latin script recognizer (covers most Western languages)
- Suspend-based API for coroutine integration
- Automatic text block + line decomposition

**MediaProjection:**
- Requires one-time user permission via PermissionActivity
- VirtualDisplay with AUTO_MIRROR flag
- Bitmap recycling to prevent memory leaks

**Content Classification:**
- Keyword-based heuristics (finance, work, personal, travel)
- Extensible pattern matching
- Returns null for unclassified content
