# Audit — registry-mind
**Reviewer: Quentin | Date: 2026-04-27 | Scope: Sprint 4+5 code**

## Health: 6.5 / 10

Architecture sound. Coroutine patterns mostly correct. Room/Flow wiring clean. Three issues must fix before release (thread safety, HTTP security, file integrity).

---

## Critical

### C1 — `RegistryIngestor.localLlm` not @Volatile — data race
`CaptureService.initLocalLlmIfReady()` writes `ingestor.localLlm` from `Dispatchers.Default`.
`processAndStore()` reads it from `Dispatchers.IO`. No synchronization. JVM may never see updated value.
**File:** `RegistryIngestor.kt:32`
**Fix:** `@Volatile var localLlm: LocalLlm? = null`

### C2 — ModelDownloadManager — no HTTP status check
`connection.responseCode` never checked. 404/503 writes HTML to .tmp file → `Result.success()`.
Service loads garbage → crash or silent LLM failure.
**File:** `ModelDownloadManager.kt:25`
**Fix:** Check `responseCode == HTTP_OK`, return `Result.failure` otherwise.

### C3 — `tmpFile.renameTo()` return value ignored
Returns `false` on cross-partition/locked-file. Code returns `Result.success(modelFile)` regardless.
**File:** `ModelDownloadManager.kt:48`
**Fix:** Check boolean, delete tmp + return `Result.failure(IOException("rename failed"))` on false.

---

## Important

### I1 — Auth token logged in production
`HttpLoggingInterceptor.Level.BODY` on all builds. `X-Registry-Auth` header in logcat on release.
**File:** `ClawConnector.kt:27`
**Fix:** `if (BuildConfig.DEBUG) Level.BODY else Level.NONE`

### I2 — `CaptureService.instance` not @Volatile
Written in onCreate/onDestroy (service thread), read in `isRunning()` from main thread.
**File:** `CaptureService.kt:40`
**Fix:** `@Volatile private var instance`

### I3 — `RegistryIngestor.scope` not cancelled in `cleanup()`
Fire-and-forget exports continue after service destroy with invalid context.
**Fix:** Add `scope.cancel()` to `cleanup()`.

### I4 — Bitmap encoded per capture but unused in local flow
Full WEBP base64 (~200KB) in every packet. `processAndStore()` ignores `imageData`. Wasted allocation.
**Fix (Sprint 6):** Lazy-encode or skip unless ExportConnector needs image.
